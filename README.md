# Handwritten Digit Recognition using TensorFlow

## Description

This project uses a neural network built with TensorFlow and Python to recognize handwritten digits from the MNIST dataset.

## Technologies

- Python
- TensorFlow
- NumPy
- Matplotlib
- Google Colab

## Results

- Training Accuracy: 98.6%
- Test Accuracy: 97%
